# English Word Rules for Karaoke Syllable Splitting

## Mục đích
Chia từ vựng tiếng Anh thành các âm tiết (syllables) đơn giản, phục vụ cho hiển thị karaoke.

---

## 0. Quy tắc đặc biệt - Từ viết tắt & Sở hữu (Apostrophe Rules)

### 0.1. Từ có dấu nháy đơn (') - KHÔNG BỊ TÁCH
Các từ chứa dấu `'` (apostrophe) như từ sở hữu, viết tắt **KHÔNG được tách** - giữ nguyên làm 1 phần:

```
✅ One's      → One's       (KHÔNG tách thành One/'s)
✅ I've       → I've        (KHÔNG tách thành I/'ve)
✅ You're     → You're      (KHÔNG tách thành You/'re)
✅ It's       → It's
✅ Don't      → Don't
✅ Can't      → Can't
✅ Won't      → Won't
✅ She's      → She's
✅ He's       → He's
✅ We're      → We're
✅ They're    → They're
✅ I'm        → I'm
✅ You've     → You've
✅ We've      → We've
✅ They've    → They've
✅ I'll       → I'll
✅ You'll     → You'll
✅ He'll      → He'll
✅ She'll     → She'll
✅ We'll      → We'll
✅ They'll    → They'll
✅ I'd        → I'd
✅ You'd      → You'd
✅ He'd       → He'd
✅ She'd      → She'd
✅ We'd       → We'd
✅ They'd     → They'd
```

**Lý do:** Trong karaoke, các từ viết tắt/sở hữu nên được hiển thị như một khối thống nhất để người xem dễ đọc và hát theo.

### 0.2. Ngoại lệ - Từ sở hữu với 's sau danh từ
Các từ sở hữu dạng `danh từ + 's` vẫn được tách theo quy tắc thông thường:
```
✅ John's     → John's      (tên riêng - giữ nguyên)
✅ Teacher's  → Teach/er's  (tách theo âm tiết, giữ 's ở cuối)
```

### 0.3. Các từ đặc biệt - fire và các từ ghép với fire
Từ `fire` và các từ ghép với `fire` được tách thành `fi/re`:
```
✅ fire        → fi/re
✅ firefighter → fi/re/figh/ter
✅ firelighter → fi/re/ligh/ter
✅ firescreen  → fi/rescreen
✅ firewater   → fi/re/wa/ter
```

### 0.4. Các từ đặc biệt khác
Một số từ có quy tắc tách đặc biệt:
```
✅ space       → space       (giữ nguyên)
✅ stat        → stat        (giữ nguyên)
✅ like        → like        (giữ nguyên)
✅ bite        → bite        (giữ nguyên)
✅ style       → style       (giữ nguyên)
```

## 1. Tiền tố (Prefixes)

### 1.1. Tiền tố `ab-`
- Nếu sau `ab` là **phụ âm** → chia `ab/...`
  ```
  ✅ ab/stract, ab/duct, ab/stinence, ab/horrent
  ```
- Nếu sau `ab` là **nguyên âm** → chia `a/b...`
  ```
  ✅ a/buse, a/bound, a/bout
  ```

### 1.2. Tiền tố `ac-`
- Nếu sau `ac` là **phụ âm khác `c`** → chia `ac/...`
  ```
  ✅ ac/creted, ac/cede
  ```
- Nếu sau `ac` là **`c` (phụ âm đôi cc)** → chia `a/cc...`
  ```
  ✅ a/cc/ession → a/ces/sion
  ```

### 1.3. Tiền tố `ad-`
- Nếu sau `ad` là **phụ âm** → chia `ad/...`
- Nếu sau `ad` là **nguyên âm** → chia `a/d...`

### 1.4. Tiền tố `a-`
- Nếu từ bắt đầu bằng `a` + phụ âm → chia sau `a`:
  ```
  ✅ a/back, a/bout, a/bove, a/bed
  ```

## 2. Hậu tố (Suffixes)

### 2.1. Hậu tố `-ted`, `-ded` tạo âm tiết riêng
Khi từ gốc kết thúc bằng `t` hoặc `d`, thì `-ed` tạo âm tiết riêng → **CHIA**:
```
✅ ac/cre/ted       (accret → kết thúc 't')
✅ ab/sent/min/ded  (absentmind → kết thúc 'd')
✅ wan/ted, nee/ded
```

### 2.2. Hậu tố `-ed`, `-s`, `-es`, `-ing` KHÔNG tạo âm tiết riêng
Khi từ gốc KHÔNG kết thúc bằng `t` hoặc `d` → **GIỮ NGUYÊN** vào phần trước:
```
✅ a/ban/doned      (abandon → kết thúc 'n', không chia -ed)
✅ a/bashed         (abash → kết thúc 'sh', không chia -ed)
✅ a/bates          (abate → kết thúc 'e', không chia -s)
✅ a/bi/ding        (abide → kết thúc 'e', không chia -ing)
```

### 2.3. Hậu tố - Phân loại theo ký tự đầu

**NGUYÊN TẮC CHUNG:**
- Hậu tố bắt đầu bằng **NGUYÊN ÂM** (a, e, i, o, u, y): TÁCH làm part riêng, phụ âm trước được "dời" vào hậu tố
- Hậu tố bắt đầu bằng **PHỤ ÂM**: Cũng TÁCH làm part riêng, nhưng không dời phụ âm

#### Hậu tố bắt đầu bằng NGUYÊN ÂM (tách + dời phụ âm trước)

| Hậu tố | Ý nghĩa | Ví dụ | Kết quả |
|--------|---------|-------|---------|
| `-er`, `-or`, `-tor` | Người thực hiện | teacher, actor, abductor | teach/**er**, ac/**tor**, ab/duc/**tor** |
| `-ee` | Người nhận hành động | employee | em/plo/**yee** |
| `-ist`, `-ian` | Người theo nghề | scientist, musician | sci/en/**tist**, mu/si/**cian** |
| `-tion`, `-sion` | Hành động, trạng thái | information, decision | in/for/ma/**tion**, de/**ci/sion** |
| `-ity`, `-ty` | Tính chất, trạng thái | activity, loyalty | ac/**ti/vi/ty**, loy/**al/ty** |
| `-ance`, `-ence` | Hành động, tình trạng | importance, difference | im/**por/tance**, dif/fe/**rence** |
| `-al`, `-ial`, `-ual` | Thuộc về | natural, editorial | na/**tu/ral**, edi/**to/ri/al** |
| `-ive`, `-sive`, `-tive` | Có tính chất | active, abusive | ac/**tive**, ab/ra/**sive** |
| `-ous`, `-eous`, `-ious` | Đầy, có tính chất | dangerous, poisonous | dan/ger/**ous**, poi/so/**nous** |
| `-ic` | Thuộc về | economic, historic | e/co/**no/mic**, his/**to ric** |
| `-ing` | Tính chất | boring, abounding | bo/**ring**, a/bound/**ing** |
| `-ize`, `-ise` | Làm cho trở thành | modernize, realize | mo/der/**nize**, re/a/**lize** |
| `-ify` | Làm cho, biến thành | beautify, identify | beau/**ti/fy**, i/de/nti/**fy** |
| `-ate` | Tạo ra, làm cho | complicate, motivate | com/pli/**cate**, mo/ti/**vate** |
| `-en` | Làm cho | shorten, strengthen | shor/**ten**, streng/**then** |
| `-able`, `-ible` | Có thể làm gì đó | comfortable, flexible | com/for/ta/**ble**, fle/xi/**ble** |

#### Hậu tố bắt đầu bằng PHỤ ÂM (tách riêng, không dời phụ âm)

| Hậu tố | Ý nghĩa | Ví dụ | Kết quả |
|--------|---------|-------|---------|
| `-ment` | Kết quả hành động | development, abandonment | de/ve/lop/**ment**, a/ban/don/**ment** |
| `-ness` | Trạng thái, phẩm chất | happiness, darkness | hap/pi/**ness**, dark/**ness** |
| `-ship` | Mối quan hệ, trạng thái | friendship, scholarship | friend/**ship**, scho/lar/**ship** |
| `-hood` | Thời kỳ, trạng thái | childhood, motherhood | child/**hood**, mo/ther/**hood** |
| `-ful` | Đầy, có nhiều | beautiful, careful | beau/ti/**ful**, care/**ful** |
| `-less` | Không có | homeless, careless | home/**less**, care/**less** |
| `-ly` | Một cách (trạng từ) | quickly, truly | quick/**ly**, tru/**ly** |
| `-ward(s)` | Hướng về phía | forward, backward(s) | for/**ward**, back/**ward(s)** |
| `-wise` | Theo hướng | clockwise, pricewise | clock/**wise**, pri/**ce/wise** |

#### Hậu tố GHÉP (attach) - không tách riêng

| Hậu tố | Ví dụ | Kết quả |
|--------|-------|---------|
| `-ve` | abuse, above | a/**buse**, a/**bove** |
| `-se` | abuse | a/**buse** |
| `-ed` (thường) | abandoned, abashed | a/ban/**doned**, a/**bashed** |
| `-s`, `-es` | abates | a/**bates** |

### 2.4. Phụ âm kép (double consonants) → TÁCH RA

Khi có **2 phụ âm giống nhau** đứng cạnh nhau → **CHIA GIỮA chúng**:

```
✅ ac/quit/tance      (phụ âm kép 'tt')
✅ ab/hor/rence       (phụ âm kép 'rr')
✅ ab/bot             (phụ âm kép 'bb')
✅ ac/cess            (phụ âm kép 'cc')
```

### 2.5. Hậu tố danh từ/tính từ KHÔNG chia nội bộ (ghép vào part cuối)
Các hậu tố sau **KHÔNG được chia nhỏ**:

| Hậu tố | Ví dụ | Kết quả |
|--------|-------|---------|
| `-nence` | abstinence | ab/sti/**nence** |
| `-lene` | abilene | a/bi/**lene** |
| `-lute` | absolute | ab/so/**lute** |
| `-tion` | ablation | a/bla/**tion** |
| `-sion` | abscission | ab/scis/**sion** |
| `-ment` | abasement | a/base/**ment** |
| `-ture` | nature | na/**ture** |
| `-sure` | pressure | pres/**sure** |
| `-ence` | reverence | rev/**erence** |
| `-ance` | acceptance | ac/cep/**tance** |
| `-tive` | active | ac/**tive** |
| `-sive` | abusive | a/bu/**sive** |
| `-tory` | history | his/**tory** |
| `-ness` | kindness | kind/**ness** |
| `-less` | careless | care/**less** |
| `-ful` | careful | care/**ful** |
| `-able` | readable | read/**able** |
| `-ible` | visible | vis/**ible** |
| `-cian` | musician | mu/**sician** |

### 2.4. Hậu tố `-tion`, `-sion`: chia TRƯỚC nếu có phụ âm + nguyên âm
Nếu ngay trước `-tion`/`-sion` có phụ âm + nguyên âm → chia trước phụ âm đó:
```
✅ ab/er/ra/tion     (trước '-tion' là 'ra')
✅ ab/scis/sion      (trước '-sion' là 'scis')
✅ a/ces/sion        (trước '-sion' là 'ces')
```

## 3. Quy tắc chia tổng quát

### 3.1. Sau phụ âm có nguyên âm → chia trước phụ âm
```
✅ a/ba/cus         (sau 'b' có 'a' → chia trước 'b')
✅ a/ban/don        (sau 'b' có 'a', sau 'd' có 'o')
✅ ab/duc/tor       (sau 'd' có 'u')
```

### 3.2. Cụm phụ âm (clusters) ở đầu âm tiết → KHÔNG chia
Các cluster phổ biến: `str`, `spr`, `scr`, `spl`, `st`, `sp`, `sc`, `sk`, `tr`, `bl`, `br`, `cl`, `cr`, `dr`, `fl`, `fr`, `gl`, `gr`, `pl`, `pr`
```
✅ ab/stract        (cluster 'str' không chia)
✅ ab/scis/sion     (cluster 'sc' không chia)
```

### 3.3. Phụ âm đôi → chia giữa
```
✅ ab/bot           ('bb' → chia giữa)
✅ ac/cess          ('cc' → chia giữa)
```

### 3.4. Từ ngắn (≤ 3 ký tự) → KHÔNG chia
```
✅ abc, abe, act
```

## 4. Các trường hợp đặc biệt (Special Cases)

Các từ sau có cách chia đặc biệt, cần hardcode:

| Từ | Kết quả | Ghi chú |
|---|---|---|
| abstinence | ab/sti/nence | -nence giữ nguyên |
| abscission | ab/scis/sion | -sion giữ nguyên |
| accreted | ac/cre/ted | -ted syllabic |
| accretion | ac/re/tion | special case |
| absentminded | ab/sent/min/ded | compound + -ded |
| accession | a/ces/sion | a- prefix + -sion |
| ablation | a/bla/tion | a- prefix + -tion |
| abilene | a/bi/lene | -lene giữ nguyên |
| absolute | ab/so/lute | -lute giữ nguyên |
| abasement | a/base/ment | -ment giữ nguyên |
| aberration | ab/er/ra/tion | ab- prefix |
| abracadabra | a/bra/ca/da/bra | từ đặc biệt |
| acetaminophen | a/ce/ta/mi/no/phen | từ y học |
| acrophobia | ac/ro/pho/bia | -phobia |
| acrostic | ac/ros/tic | special |
| acquiescence | ac/qui/es/cence | -quiescence |
| acquisitive | ac/qui/si/tive | -quisitive |
| acquire | ac/quire | -quire giữ nguyên |
| abidance | a/bi/dance | -dance giữ nguyên |
| abiding | a/bi/ding | -ding giữ nguyên |

## 5. Thứ tự xử lý

1. **Kiểm tra từ ngắn** (≤ 3 ký tự) → giữ nguyên
2. **Kiểm tra special cases** → trả về kết quả hardcode
3. **Nhận diện hậu tố**:
   - **Separate suffixes** (`-tion`, `-sion`, `-ment`, `-cian`):
     - Tách hậu tố làm part riêng
     - Phần gốc tiếp tục chia
   - **Syllabic `-ted`/`-ded`** (gốc kết thúc `t`/`d`):
     - Tách `ed` làm part riêng
   - **Attach suffixes** (`-ed`, `-s`, `-es`, `-ing`, `-nence`, `-lene`, `-lute`, `-ture`, `-sure`, `-ence`, `-ance`, v.v.):
     - Giữ nguyên, ghép vào part cuối
4. **Nhận diện tiền tố** (`ab-`, `ac-`, `ad-`, `a-`)
5. **Chia phần rest**: sau phụ âm có nguyên âm → chia trước phụ âm (trừ clusters)
6. **Ghép hậu tố**:
   - Separate → thêm part riêng
   - Attach → ghép vào part cuối
